Large integer multiplier research, University of Toronto

Overview

Build a very large hardware multiplier, with input size of 2048 bits. The reason why this multiplier is needed remains a mystery.

Description

For this research, most of the designs are implemented in systemVerilog. This is my first time writing systemVerilog, and it was not so bad as the syntax is similar to Verilog on which I had lots of experience. Besides the design files, I familiarized myself with wrting testbenches so that I can verify the correctness of my design on ModelSim. The research project also gave me a chance to practice my Python skills as I needed to write software model for my hardware design of the multipliers.

Computer vulnerability detection, University of Toronto

Overview

Compile open source projects with clang and LLVM to extract LLVM bitcode for employing machine learning in vulnerability detection.

Description

For this project, the major challenge is that every different open source project has its own way of project organization. In order to extract the LLVM bitcode, I need to find the correct configuration of the project so that I can use llvm to compile it.
