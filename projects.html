Projects

Hardware Sokoban

This is a project created for the course Digital Systems. Sokoban is a classic game in which the player pushes the boxes to the targets while bypassing obstacles in the game area. This version of the game is implemented using Verilog. The player controls the game with PS2 keyboard.



Minesweeper

Minesweeper game created with Pygame. Classic of the classics, not much to explain, except ALL the images and sprites are hand-drawn!



Nox Mapper

A mapping application for courier path routing. This is a project for the course Software Design and Communication. The map features not only routing function but also night mode and on-map distance measurements.Graphical user interface is implemented using EZGL graphical libraries.



Stroop Effect Game

This is game to test Stroop effect. Designed as the final project for the course Computer Organization, this game is implemented on FPGA DE1-SoC, with some of the code written in assembly.
